# 艾尔德兰魔法体系

> *「魔法不是力量，是理解——理解世界的运行方式，然后轻轻推它一把。」*
> *——大魔导师·梅林达·银叶*

---

## 魔法的本质

在艾尔德兰，魔法被称为**"以太"**（Aether）——一种弥漫在整个世界中的无形能量。万物皆以太，但只有少数生命能够感知和引导它。

**魔力的来源**：
- **环境以太**：弥漫在空气、水、土壤中的游离魔力。在某些特殊地点（如世界树遗迹、龙眠谷）浓度极高。
- **生命以太**：每个生物体内都含有微量的魔力。有天赋者体内浓度远高于常人。
- **魔石**：以太凝结形成的矿物，是炼金和附魔的核心材料。

**魔力的流动**：魔力在世界中循环流动，如同潮汐。满月时魔力潮汐最强，新月时最弱。这也是为什么某些强大的仪式必须在特定时间进行。

## 天赋与觉醒

### 魔法天赋

并非所有人都能使用魔法。大约每 100 个人中只有 3-5 人具有可被培养的魔法天赋。天赋的高低取决于体内以太的浓度和感知能力。

**天赋等级**：
| 等级 | 描述 | 人口占比 |
|------|------|----------|
| 无 | 完全无法感知魔力 | ~95% |
| 微弱 | 能感知魔力波动，但极难引导 | ~3% |
| 普通 | 可以学习基础魔法 | ~1.5% |
| 优秀 | 学习速度较快，上限较高 | ~0.4% |
| 卓越 | 天生的魔法感应者，几乎不需要训练 | ~0.05% |
| 传说 | 千年一遇，魔力如泉涌不息 | 极罕见 |

**天赋测试**：银塔学院使用"水晶蝴蝶"进行天赋测试——蝴蝶会根据被测试者的魔力浓度改变颜色和飞行轨迹。

**觉醒**：有些人的魔法天赋会在幼年自发觉醒，表现为"魔力爆发"（`magic_burst_baby`）。这通常发生在强烈情绪波动时（恐惧、愤怒、悲伤）。幼年觉醒的儿童会被学院主动联系（`magic_academy_notice`）。

## 魔法学派

银塔学院将魔法分为**三大正统学派**和**一个禁忌学派**：

### 元素系（Elemental）

最基础也最常用的魔法体系。通过引导以太来操控自然元素。

**子分类**：
- **火**：火焰投射、火墙、爆裂术、陨石术
- **冰**：冰锥、冰冻、暴风雪、绝对零度
- **雷**：闪电、电弧、雷暴、天罚
- **风**：风刃、加速、飞行、龙卷风
- **土**：岩石投射、地裂、石墙、地震

**特点**：攻击力强、效果直观、学习曲线平缓。大多数冒险者会学习基础的元素魔法。

**代表人物**：骑士团长·加雷斯（精通火系辅助魔法）

### 灵魂系（Soul）

最精密也最深奥的魔法体系。直接作用于生命体的灵魂层面。

**子分类**：
- **治愈**：伤口修复、解毒、净化、复活（极高级）
- **增益**：力量增幅、速度增幅、魔力增幅、护盾
- **预言**：千里眼、预知梦（`spr_dream_vision`）、占卜、命运窥探
- **灵魂术**：灵魂出窍、灵体交流、灵魂绑定（`soul_bound`）

**特点**：需要极高的感知力和精神力，学习难度大。但高级灵魂系魔法的效果是其他学派无法替代的。

**代表人物**：大魔导师·梅林达（据说精通所有灵魂系魔法）

### 自然系（Nature）

与自然万物沟通的魔法体系。通过以太与动植物建立共鸣。

**子分类**：
- **植物**：操控植物生长、藤蔓束缚、毒孢子、生命汲取
- **动物**：与动物沟通、动物控制、兽化术
- **变形**：改变自身形态（需要极高的魔力控制）
- **炼金**：药水调配、物质转化（`luk_potion_find`）

**特点**： versatility 强，适合辅助和生存。矮人的锻造术与炼金有相通之处。

**代表人物**：翡翠密林的森林精灵（虽然已几乎灭绝）

### 炉纹锻造（Runic Craft）——非学院正统分类

严格来说，银塔学院并不把矮人的工艺传统单独列为一个正式学派，因为它不像元素系那样依赖施术者即时引导以太，也不像灵魂系那样直接作用于生命本身。但在实务层面，矮人的炉纹、附铭、主炉调火和魔矿精炼，已经构成了一套稳定而古老的施作体系。

**核心原理**：
- **魔矿导性**：炽焰山脉的魔矿能储存、传导并缓慢释放以太，是矮人工艺的物质基础
- **炉火调谐**：矮人相信"龙息即炉火"。主炉的温度、风箱节奏与锻打顺序，会直接影响魔矿中的以太稳定性
- **炉纹与铭文**：炉纹并非传统意义上的咒文，更接近用金属、石材和刻线固定以太流向的结构语言
- **工序记忆**：许多高阶工艺依赖师徒口授与手感传承，写下来的只是骨架，真正的诀窍在火候、回音和落锤轻重之间

**主要用途**：
- 锻造具备高耐久、元素抗性的兵器与防具
- 为城门、矿井、遗迹通道和封锁机构提供长期稳定的动力与结构支撑
- 加固或修复龙族时代遗留下来的机关、熔炉与封印设施

**局限性**：
- 几乎无法像学院法师那样即时施法，更依赖工坊、材料与完整工序
- 对个人 mag 的要求往往低于法师体系，但对经验、纪律和材料控制要求极高
- 一旦材料纯度、炉温或纹路顺序出错，轻则作品报废，重则引发爆裂、塌炉或矿脉失衡

### 暗影系（Shadow）——禁忌

被银塔学院和圣光教会共同禁止的魔法体系。通过扭曲以太来达到目的，代价是使用者的灵魂和理智。

**子分类**：
- **诅咒**：削弱、厄运、痛苦、灵魂束缚（`lover_curse`）
- **亡灵术**：操控尸体、召唤亡灵、灵魂囚禁
- **心灵术**：读取记忆、植入思想、精神控制
- **深渊术**：召唤魔界生物、虚空之力、空间撕裂

**代价**：每次使用暗影魔法，使用者的灵魂都会受到侵蚀。长期使用会导致：
1. 记忆混乱
2. 情感麻木
3. 外貌异变（瞳孔变黑、皮肤苍白）
4. 最终灵魂破碎，变成"空壳"——一具只服从本能的躯壳

**禁忌学院**认为这些代价是可以接受的——他们正在研究"逆转灵魂侵蚀"的方法，但至今没有成功。

**与玩家相关**：
- `dark_mage_choice`：选择学习暗影魔法
- `lover_curse`：被暗影诅咒
- `spr_curse_breaker`：解除诅咒
- `world_breaking_start`：暗影势力引发的世界危机

## 魔法阶位

银塔学院将魔法能力分为 0-10 阶：

| 阶位 | 名称 | 能力描述 |
|------|------|----------|
| 0 | 启蒙 | 能感知魔力，但无法释放 |
| 1 | 初学 | 能释放最基础的魔法（小型火球、光球） |
| 2 | 入门 | 能熟练使用 2-3 个基础法术 |
| 3 | 进阶 | 能释放中级魔法（`magic_duel` 中"已能使用4阶"） |
| 4 | 精通 | 能在战斗中灵活运用魔法 |
| 5 | 大师 | 能创造变体法术，教学资格 |
| 6 | 宗师 | 魔法造诣达到学院教授级别 |
| 7 | 大魔导师 | 能独自完成高级法术研究 |
| 8 | 贤者 | 名字被记录在魔法史中 |
| 9 | 半神 | 据说可以触及世界的本质 |
| 10 | ？？？ | 从未有记载 |

## 魔法与属性的关系

| 属性 | 魔法影响 | 事件体现 |
|------|----------|----------|
| **mag（魔力）** | 魔法总量、法术威力 | 高 mag → 能学习更高级的魔法 |
| **int（智力）** | 魔法学习速度、法术精度 | 高 int → 考试更容易过、法术更精准 |
| **spr（灵魂）** | 精神力、抗精神攻击、感知力 | 高 spr → 预言能力、灵魂系天赋、抗诅咒 |
| **str（体魄）** | 魔法战斗时的体能 | 影响施法的持续时间 |
| **luk（幸运）** | 魔法暴击、意外发现 | `luk_potion_find`、`luk_wild_encounter`、`luk_lottery` |

## 特殊现象

### 魔法共鸣

当两个灵魂系魔法师的魔力频率一致时，会发生"共鸣"——双方魔力互相增幅，能释放远超个人极限的魔法。这就是"灵魂绑定"（`soul_bound`）的魔法原理。

### 灵魂契约

一种古老的契约形式，将两个灵魂以魔法链接在一起。好处是双方共享感知和魔力，坏处是一方受伤另一方也会承受痛苦。

### 矮人的魔法抗性

矮人的"魔法抗性"并不是完全免疫法术，而更像一种长期与魔矿、炉火和龙息残痕共处后形成的稳定性优势。

- 对**元素系冲击**的耐受通常高于人类，尤其擅长承受高温、爆震和金属变形类影响
- 对**附魔装备**和炉纹结构的适配度较高，能长期使用高负荷工艺环境而不易崩溃
- 对**灵魂系与精神系**影响并无天然豁免，因此教会与诅咒术对矮人依然有效
- 部分学者认为，这种抗性同时来自体质、工艺训练和炽焰山脉环境，而不只是血统本身

### 禁术·复活

理论上，灵魂系最高级的治愈术可以将刚刚死去（灵魂尚未完全脱离）的人复活。但实际上成功概率极低，且代价巨大——施术者需要消耗自己一部分灵魂作为代价。圣光教会将此列为"极度禁忌"，只有教皇才有权决定是否执行。
