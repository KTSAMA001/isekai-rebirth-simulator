## 王国势力

### 艾尔德王室

**象征**：银色王冠与交叉双剑
**首领**：艾尔德国王（当前：阿尔弗雷德七世）
**驻地**：银冠城·王城

艾尔德王国建立于千年前魔导战争胜利之后，由人类英雄·建国者艾尔德一世创立。王室以"银冠"为徽记，传承千年，历经数次王朝更替（通过继承和篡位），但国号始终不变。

**内部结构**：
- **国王**：最高统治者，拥有军事和立法权
- **贵族议会**：由各领地主组成的议事机构，对国王的征税和征兵有一定制约力
- **平民代表**：近年来新设的机构，由各城镇选出的代表组成，权力有限

**与玩家相关**：
- `birth_noble` / `birth_noble_estate`：出生为贵族，自动获得贵族 flag
- `become_lord`：通过功绩或财富获得封地，成为领主
- `mid_political_intrigue`：卷入宫廷阴谋
- `chain_rise_to_power`：从平民崛起，进入权力圈

### 领主阶层

王国实行分封制。国王将领土分封给贵族，贵族再将土地分封给骑士。每个领主在自己的领地上拥有税收、审判和征兵权。

**等级**：
1. **公爵**：王国最显赫的贵族，通常统治一个行省
2. **伯爵**：统治一座主要城市及周边地区
3. **子爵**：统治一个城镇
4. **男爵**：统治数个村庄
5. **骑士**：最低等级的贵族，拥有一小块封地

## 宗教势力

### 圣光教会

**象征**：金色太阳
**首领**：教皇（当前：艾薇琳三世）
**驻地**：银冠城·圣光大教堂

大陆上影响力最大的宗教组织。信仰"圣光之神"——据说在魔导战争中引导人类取得胜利的至高存在。

**教义核心**：
- 圣光是唯一的正义之源
- 魔法是"圣光的碎片"，应当被合理使用，而非被崇拜
- 暗黑魔法是"堕落之光"，必须被根除
- 帮助弱者是每个信徒的义务

**主要机构**：
- **圣光大教堂**：信仰中心，也是教堂孤儿院的所在地
- **审判庭**：负责追捕异端和暗黑魔法师，拥有独立司法权
- **治愈者修道会**：专门训练治愈魔法的修女和修士
- **远征骑士团**：教会的武装力量

**与玩家相关**：
- `church_orphan`：在教会孤儿院长大
- `demon_hunt`：参与教会组织的魔物讨伐
- `spr_curse_breaker`：被教会请求解除诅咒
- `knight_siege`：教会参与保卫战

**暗面**：教会并非纯粹的善。审判庭的手段有时过于残酷，对"异端"的定义随时代变化。部分高级神职人员暗中研究被禁止的灵魂操控术。

## 学术势力

### 银塔魔法学院

**象征**：银色高塔与星辰
**院长**：大魔导师·梅林达（精灵混血，据传已超过三百岁）
**驻地**：银冠城·学院山

大陆上最权威的魔法教育机构，建于王国建立之初。学院依山而建，由七座相连的高塔组成，每座塔代表一个魔法学科。

**塔楼分配**：
| 塔楼 | 学科 | 颜色 |
|------|------|------|
| 红塔 | 元素魔法（火、冰、雷） | 红色 |
| 蓝塔 | 灵魂魔法（治愈、增益、预言） | 蓝色 |
| 绿塔 | 自然魔法（植物、动物、变形） | 绿色 |
| 紫塔 | 空间魔法（传送、结界、储物） | 紫色 |
| 金塔 | 附魔与炼金 | 金色 |
| 灰塔 | 防御与反魔法 | 灰色 |
| 黑塔（禁忌） | 暗黑魔法研究 | 被封印 |

**入学与毕业**：
- 入学年龄通常 10-13 岁，需要通过魔力资质测试（`magic_academy_notice`）
- 学制 7-10 年，通过毕业考核后获得法师资格（`magic_graduate`）
- 学院每年举办决斗赛（`magic_duel`），排名影响毕业分配

**黑塔·禁忌藏书区**：
- 学院地下深处，入口刻有警告铭文
- 藏有被禁止的魔法书籍，包括暗影系和灵魂禁术
- `mage_arcane_library`：学生发现并进入禁忌区
- `dark_mage_choice`：学生被暗黑魔法诱惑

**与玩家相关**：
- `magic_academy_notice` → `magic_exam` → `magic_duel` → `magic_graduate`：完整学院路线
- `mage_arcane_library` → `dark_mage_choice`：暗黑魔法分支
- `mage_magic_tower`：参观/拜访著名法师塔
- `magic_breakthrough_final`：成为传奇法师的终极突破
- `master_spell`：创造出属于自己的奥术

## 军事势力

### 铁冠骑士团

**象征**：铁冠与剑盾
**团长**：骑士团长·加雷斯·铁心
**驻地**：银冠城·骑士堡

王国的正规军事力量，由最精锐的骑士组成。骑士团直接效忠于国王，不受领主节制。

**晋升体系**：
1. **侍从**（`squire_training`）：跟随骑士学习基础剑术和礼仪
2. **见习骑士**：通过骑士考核（`knight_examination`）
3. **正式骑士**：完成首次任务（`first_quest`）
4. **资深骑士**：多次立功
5. **骑士长**：统帅一队骑士
6. **骑士团长**：骑士团最高指挥官

**骑士信条**：
- 荣誉高于生命
- 保护弱者，服从正义
- 永不背叛誓言
- 勇敢面对恐惧

**与玩家相关**：
- `squire_training` → `knight_examination` → `first_quest`：骑士路线
- `knight_tournament` → `knight_glory`：骑士比武和荣耀
- `knight_siege`：保卫城堡
- `knight_dragon_quest`：讨伐巨龙的骑士任务
- `challenge_final_boss`：最终决战

## 经济势力

### 贸易者公会

**象征**：天平与金币
**会长**：商会会长（每五年选举一次）
**驻地**：河湾镇·商会大厦

大陆上最大的商业组织，控制着大部分长途贸易路线。虽然名义上是"公会"，实际权力堪比小国。

**组织结构**：
- **商人学徒**（`merchant_apprentice`）：跟随老商人学习
- **行商**：获得执照的独立商人
- **商会会员**：有一定资本和贸易网的商人
- **商会理事**：掌管某条贸易路线
- **商会会长**：最高权力者

**业务范围**：
- 长途贸易（`mny_trade_route`）：东方丝绸、南方香料、北方毛皮、西方海鲜
- 拍卖行（`merchant_auction`）：稀有物品的交易场所
- 金融服务：放贷、保险、货币兑换
- 情报网络：商会在每个城镇都有眼线

**与玩家相关**：
- `merchant_apprentice` → `merchant_career`：商人路线
- `mny_trade_route` → `mny_tax_crisis` → `merchant_economic_crisis`：商业挑战
- `merchant_guild` → `become_lord`：商业帝国 → 领主
- `mid_business_rivalry`：商业竞争
- `investment_opportunity`：投资机会

## 冒险者势力

### 冒险者公会

**象征**：交叉的剑与盾（公会标志），各城镇分会有不同的编号门牌
**总部会长**：传奇冒险者·"灰发"阿尔冯斯（据说70岁仍在冒险）
**总部驻地**：银冠城

一个松散但有效的组织，为冒险者提供任务发布、报酬结算和基本保障。

**等级制度**：

| 等级 | 徽章 | 接受任务范围 |
|------|------|-------------|
| F 级 | 铁牌 | 寻物、送信、打扫 |
| E 级 | 铜牌 | 采集、护送、清剿小怪 |
| D 级 | 银牌 | 地城浅层探索、中型魔物讨伐 |
| C 级 | 金牌 | 地城中层探索、大型魔物讨伐 |
| B 级 | 白金牌 | S级任务、精英地城 |
| A 级 | 钻石牌 | 传说级任务、国家级委托 |
| S 级 | 传说之证 | 仅授予完成传奇事迹者 |

**与玩家相关**：
- `guild_join`：加入冒险者公会
- `first_quest` → `adv_bounty` → `adv_uncharted` → `adv_legendary_dungeon`：冒险者路线
- `bounty_notice`：接受悬赏任务
- `advanced_dungeon`：挑战高难度地城
- `lost_in_dungeon`：地城遇险

## 隐藏势力

### 暗影兄弟会

**象征**：无（信奉"不存在之物"）
**首领**：未知
**驻地**：传说中遍布大陆各主要城市的地下网络

大陆上最大的犯罪组织，从事暗杀、走私、情报贩卖和地下赌博。教会和骑士团多次试图剿灭，但始终未能成功。

**运作方式**：
- 没有固定成员名册，通过暗号和中间人联络
- 在每个城市都有"影子"——隐藏的据点
- 接受一切委托，只要有足够的报酬
- 据说有成员渗透进了贵族议会和教会

**与玩家相关**：
- `chain_dark_past`：发现主角的黑暗过去与此有关
- `mid_political_intrigue`：暗影兄弟会可能参与政治阴谋
- `dual_merchant_spy`：商业间谍活动

### 禁忌学院

**象征**：破碎的面具
**创始人**：传说中的"堕落大魔导师"·维克托
**驻地**：未知（传说在地下深处的某个维度裂缝中）

一个秘密组织，致力于研究和使用被银塔学院禁止的暗黑魔法。成员多为被学院驱逐或主动离开的魔法师。

**核心理念**：
- 魔法没有善恶，只有强弱
- "圣光"不过是另一种形式的魔力
- 追求魔法的终极形态——"超越"

**与玩家相关**：
- `dark_mage_choice`：选择走上暗黑魔法之路
- `lover_curse`：暗黑诅咒
- `challenge_final_boss`：最终决战可能与暗影势力有关
- `world_breaking_start`：世界危机的幕后推手之一

## 势力关系网

```
        王室
       ╱    ╲
  骑士团    教会
     │       │
     │   审判庭
     │       │ (追捕)
     │       ▼
  商会 ◄── 暗影兄弟会
     │
  冒险者公会
     │
  银塔学院
     │ (驱逐/禁止)
     ▼
  禁忌学院
```

**同盟**：王室 ↔ 骑士团、王室 ↔ 教会（表面）、商会 ↔ 冒险者公会
**敌对**：教会 ↔ 禁忌学院、骑士团 ↔ 暗影兄弟会、审判庭 ↔ 暗影兄弟会
**中立**：商会 ↔ 各方（只看利益）、冒险者公会 ↔ 各方（谁付钱帮谁）
